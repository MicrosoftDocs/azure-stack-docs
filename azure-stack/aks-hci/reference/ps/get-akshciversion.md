# Get-AksHciVersion

## Synopsis
Get the current Kubernetes version of Azure Kubernetes Service on Azure Stack HCI and Windows Server.

## Syntax

### Get current version
```powershell
Get-AksHciVersion  
                          
```

## Description
Get the current Kubernetes version of Azure Kubernetes Service on Azure Stack HCI and Windows Server.

```powershell
Get-AksHciVersion
```
