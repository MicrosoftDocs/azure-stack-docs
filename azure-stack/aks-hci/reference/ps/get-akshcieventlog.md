# Get-AksHciEventLog

## Synopsis
Gets all the event logs from the Azure Kubernetes Service on Azure Stack HCI and Windows Server PowerShell module.

## Syntax

```powershell
Get-AksHciEventLog
```

## Description
Gets all the event logs from the Azure Kubernetes Service on Azure Stack HCI and Windows Server PowerShell module.

## Examples

### Example
```powershell
Get-AksHciEventLog
```
