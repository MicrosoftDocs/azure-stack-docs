# Disable-AksHciOfflineDownload

## Synopsis

Disables offline downloading to get the AKS on Azure Stack HCI images.

## Syntax

```powershell
Disable-AksHciOfflinedownload
```

## Description

Disables offline downloading to get the AKS on Azure Stack HCI images.

## Examples

```PowerShell
Disable-AksHciOfflinedownload
```
