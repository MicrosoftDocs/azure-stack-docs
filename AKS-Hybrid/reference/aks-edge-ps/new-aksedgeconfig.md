# New-AksEdgeConfig

Creates a JSON file with the configurations.

## Syntax

```powershell
New-AksEdgeConfig [[-outFile] <String>]
```

## Description
Creates a sample configuration file needed to create an AKS Edge deployment.


## Parameters

### -outFile
Provide the name of configuration file
