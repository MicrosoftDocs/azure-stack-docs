# Get-AksEdgeDeploymentInfo

Gets AksEdge deployment information.

## Syntax

```powershell
Get-AksEdgeDeploymentInfo
```

## Description

Gets AksEdge deployment returns networking, node and Arc connectivity status and name for the cluster of the local AksEdge deployment.
