# Get-AksEdgeNodeConnectivityMode

Gets the connectivity mode of the AKS Edge Essentials Linux node (ssh access on or off). Returns `On` when the ssh access is enabled.

## Syntax

```powershell
Get-AksEdgeNodeConnectivityMode 
```

## Description

Gets the connectivity mode of the AKS Edge Essentials Linux node (ssh access on or off). Returns `On` when the ssh access is enabled.


## Examples

```powershell
Get-AksEdgeNodeConnectivityMode 
```
