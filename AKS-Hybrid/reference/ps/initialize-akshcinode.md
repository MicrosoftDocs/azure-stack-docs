# Initialize-AksHciNode

## Synopsis
Run checks on every physical node to see if all requirements are satisfied to install Azure Kubernetes Service on Azure Stack HCI and Windows Server.

## Syntax

```powershell
Initialize-AksHciNode
```

## Description
Run checks on every physical node to see if all requirements are satisfied to install Azure Kubernetes Service on Azure Stack HCI and Windows Server.

## Examples

### Example
```powershell
Initialize-AksHciNode
```
