# Uninstall-AksHciCsiNfs

## Synopsis
Uninstalls CSI NFS Plugin in a cluster.

## Syntax

```powershell
Uninstall-AksHciCsiNfs -name <String>                       
```

## Description
Uninstalls CSI NFS Plugin in a cluster.

## Examples

### Example

```PowerShell
Uninstall-AksHciCsiNfs -name mycluster
```

## Parameters

### -name
The alphanumeric name of your Kubernetes cluster.
