# Uninstall-AksHciAdAuth

## Synopsis
Uninstall Active Directory authentication.

## Syntax

```powershell
Uninstall-AksHciAdAuth -name <String>
```

## Description
Uninstall Active Directory authentication.

## Examples

### Example
```powershell
Uninstall-AksHciAksHciAdAuth -name myCluster
```

## Parameters

### -name
The alphanumeric name of your Kubernetes cluster.
