# Install-AksHciGmsaWebhook

## Synopsis
Installs gMSA webhook add-on to the cluster.

## Syntax

```powershell
Install-AksHciGmsaWebhook -name <String> 
                          [-activity <String>]                      
```

## Description
Installs gMSA webhook addon to the cluster.

## Examples

### Example

```PowerShell
Install-AksHciGmsaWebhook -name mycluster
```

## Parameters
